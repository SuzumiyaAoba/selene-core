# selene-core

Moonbit で Scheme (R5RS) を実装するためのリポジトリです。まずは R5RS の完全実装と最小限の REPL を目指します。

## 必要環境
- MoonBit CLI（最新版を推奨）
- make（任意。用意した簡易タスクを実行する場合に利用）

## セットアップと利用方法
1. MoonBit CLI をインストールし、`moon --version` で動作を確認してください（インストール方法は公式ドキュメントを参照）。
2. 依存取得とビルド: `make build` または `moon build`
3. テスト実行: `make test` または `moon test`（テストはこれから追加します）
4. サンプル実行: `make run` または `moon run`

## ディレクトリ構成
- `src/` : 本体実装。現状はエントリーポイントのスタブのみ。
- `tests/` : テスト用ディレクトリ（これから拡充）。
